# 固醇
## Intro
固醇則是細胞膜的重要成分。

## 分類

> [!NOTE]
> 可以按照來自於哪裡，分為動物固醇和植物固醇。

固醇

| 母項目 | 種類 | 來自於 |
| - | - | - |
| 固醇 | 動物固醇 | 動物 |
| 固醇 | 植物固醇 | 植物 |

動物固醇

| 母項目 | 種類
| - | - |
| 動物固醇 | 膽固醇 | 

植物固醇

| 母項目 | 種類 | 存在於 |
| - | - | - |
| 植物固醇 | 菜油甾醇 | 蔬果、果實、種子 |
| 植物固醇 | 谷固醇 | 顫楊木 |
| 植物固醇 | 麥角固醇 | 真菌細胞膜
